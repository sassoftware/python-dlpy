dlpy.model.Model.plot_training_history¶

Model.plot_training_history(items=['Loss', 'FitError'], fig_size=(12, 5), tick_frequency=1)¶

Display the training iteration history. If using in Jupyter, supress return object with semicolon - plot_training_history();

Parameters:
items : list, optional

Specifies the items to be displayed. Default : [‘Loss’, ‘FitError’)

fig_size : tuple, optional

Specifies the size of the figure. Default : (12, 5)

tick_frequency : int, optional

Specifies the frequency of the ticks visable on xaxis. Default : 1
Returns:
matplotlib.axes.Axes
